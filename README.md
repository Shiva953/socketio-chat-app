# socketio-chat-app
Real time chat application built using NodeJS and SocketIO.

## How to run
1. Clone this repository to your local machine.
2. Navigate to the server directory: cd socketio-chat-app/server
3. Install the dependencies: npm install
4. Start the server by typing node index.js
This would start the server on localhost:3000.

Navigate to localhost:3000 on your web browser to access the application.

The root directory contains two subdirectories: client and server. The client directory contains the client-side code and the server directory contains the server-side code.
